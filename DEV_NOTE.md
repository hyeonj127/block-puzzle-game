# BLOCK BLAST - 구현 기능 및 개발 노트 (DEV_NOTE.md)

> **문서 목적:** 블록 블라스트 웹 버전 개발 과정에서 추가/개선한 핵심 기능들을 추적하고 검증하기 위한 개발자용 기록 문서입니다.

---

## 주요 업데이트 및 개선 현황

### 1. 조작감 및 드래그 반응성 강화

* **즉시 반응 드래그:** 블록을 드래그할 때 `requestAnimationFrame`으로 한 프레임 늦게 반영하던 부분을 제거하고, 마우스/터치 이동 이벤트가 들어오는 즉시 블록 위치가 갱신되도록 수정했습니다.
* **거리 기반 드래그 가속:** 드래그 시작점에서 손가락이 멀어질수록 이동 배율이 부드럽게 증가하며, 최대 `1.65배`까지 빨라지도록 적용했습니다. 드롭 판정은 가속 후 실제 블록 위치를 기준으로 처리합니다.
* **완성 줄 미리보기:** 드래그 중 현재 위치에 블록을 배치했을 때 완성되는 가로줄과 세로줄 전체를 금색 펄스 효과로 강조합니다.
* **하드웨어 가속 유지:** CSS `translate3d`를 사용해 GPU 가속을 유지하며, 드래그 중 블록 이동이 부드럽게 보이도록 했습니다.

### 2. 손가락 가림 방지 오프셋 구현

* **시야 확보:** 모바일에서 블록이 손가락에 가려지지 않도록 드래그 중 블록을 손가락 기준 `100px` 위로 표시합니다.
* **배치 좌표 보정:** 실제 배치 판정 좌표도 같은 오프셋 기준으로 보정해, 사용자가 보는 위치와 배치되는 위치가 어긋나지 않도록 했습니다.
* **드롭 스냅 보정:** 블록을 놓은 위치가 칸 중심에서 조금 어긋나도 약 `1.15칸` 반경 안의 가장 가까운 유효 위치를 찾아 자동으로 배치합니다.

### 3. 게임 규칙 반영: 점수, 리필, 줄 제거

* **8x8 보드:** 게임 맵은 8x8 블록 보드판입니다.
* **블록 리필:** 블록은 3개가 랜덤으로 제공되며, 3개를 모두 배치하면 새 블록 3개가 리필됩니다.
* **기본 배치 점수:** 블록 조각 1개를 배치할 때마다 `1점`을 즉시 얻습니다.
* **줄 제거 점수:** 한 번에 `n`줄을 제거하면 기본 점수로 `10 * n^2`점을 얻습니다.
* **줄 제거 규칙:** 가로줄 또는 세로줄이 꽉 차면 해당 줄의 블록이 제거됩니다.

### 4. Classic 콤보 시스템

* **Classic 모드:** 점수를 최대한 많이 올리는 것이 목표이며, 줄을 제거할 때마다 콤보가 증가합니다.
* **콤보 배수:** Classic 모드에서는 콤보 수가 줄 제거 점수의 배수로 적용됩니다.
* **콤보 유지 조건:** 콤보가 끊길 상황이어도, 같은 3블록 턴 안에서 한 블록으로 2줄 이상 제거하면 콤보가 이어집니다.

### 5. 올 클리어 맵 테마 변경

* **올 클리어 보상:** 모든 블록을 없애 보드가 비면 보너스 점수를 얻고, 맵 테마가 다음 테마로 변경됩니다.
* **테마 클래스:** 보드에 `theme-0`부터 `theme-3`까지의 클래스를 적용해 배경, 테두리, 빈 칸, 채워진 블록 색상이 함께 바뀌도록 했습니다.
